This document is my focused roadmap toward becoming a **Backend Engineer (with deployment knowledge)** and landing an internship. The goal is to stay focused and not just randomly learn.

---

# Technology Roadmap (Level-Wise)

Everything is divided into 3 levels:

- **Level 1** → Internship Ready
- **Level 2** → Strong Backend Candidate
- **Level 3** → Standout / Advanced

Each section includes: Estimated Time to Learn · Where to Implement · How to Showcase

---

# LEVEL 1 — Core Backend (Internship Ready)

These are non-negotiable.

## 1. HTTP + REST API Design

- Status codes
- Idempotency
- CRUD design
- Request validation
- Error handling

**Implementation:** SafeShare project  
**Showcase:** Clean Swagger docs + proper endpoint design

---

## 2. PostgreSQL + SQL

- SELECT, JOIN, GROUP BY
- Indexes
- Transactions
- Normalization

**Implementation:** SafeShare, any CRUD project  
**Showcase:** Schema diagrams + explain analyze results

---

## 3. Authentication & Authorization

- JWT (stateless auth)
- bcrypt (password hashing)
- RBAC (Role-Based Access Control)

**Implementation:** SafeShare Backend  
**Showcase:** Auth flow diagram + API docs

---

## 4. Docker

- Dockerfile
- docker-compose
- Volumes + networking
- Multi-stage builds

**Implementation:** Every project containerized  
**Showcase:** Public Docker Hub images or Dockerfile in repo

---

## 5. Git & GitHub Workflow

- Branching
- PRs and reviews
- Conventional commits
- CI basics

**Implementation:** Daily usage  
**Showcase:** Clean commit history, good READMEs

---
